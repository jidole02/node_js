# express란? 

----

- 개념 : **node js** 를 사용하여 쉽게 서버를 구성할 수 있게 만든 클래스와 라이브러리 집합체이다.

- express의 사용!

  ```javascript
  const app = express()
  // app에 express담아 와서
  const port = 3000
  
  app.get('/', (req, res) => {
    res.send('Hello World!')
  })
  
  // app.use => 사용하겠다...
  // app.listen => 서버를 열겠다...
  app.listen(port, () => {
    console.log(`Example app listening at http://localhost:${port}`)
  })
  ```

  

- express를 사용하지 않고 개발하려면 굉장히 고단하다. 그니까 잘 쓰자...
